🛒 Real Mart Sales Analysis Dashboard

 Project Title / Headline

**Real Mart Sales Analysis Dashboard: Retail Sales Performance Analysis Using Microsoft Excel**

An interactive Excel dashboard designed to analyze sales performance, profitability, customer behavior, branch performance, and payment preferences across multiple cities. The dashboard enables stakeholders to monitor key business metrics and make informed decisions through data-driven insights.

---

## Short Description / Purpose

The Real Mart Sales Analysis Dashboard provides a comprehensive overview of retail sales operations using Excel's analytical and visualization capabilities.

The dashboard helps business managers, sales teams, and decision-makers track sales trends, evaluate branch performance, analyze customer preferences, and identify opportunities for revenue growth and operational improvement.

---

# Features / Highlights

## Business Problem

Retail businesses often struggle to monitor sales performance across multiple branches and cities due to scattered data and manual reporting processes.

Management needs answers to critical questions such as:

* Which city generates the highest revenue?
* Which branch is performing best?
* Which product categories drive the most sales?
* How are sales changing over time?
* Which payment methods do customers prefer?
* Are customers satisfied with their shopping experience?

Without a centralized reporting system, identifying trends and making strategic decisions becomes difficult.

---

## Solution

To address these challenges, an interactive Excel dashboard was developed that consolidates sales data into a single view.

The dashboard enables stakeholders to:

* Monitor total sales and profit.
* Analyze sales trends over time.
* Compare branch and city performance.
* Identify top-performing product categories.
* Understand customer payment preferences.
* Track customer satisfaction ratings.
* Perform dynamic analysis using slicers and filters.
